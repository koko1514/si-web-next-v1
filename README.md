# 🎓 Sistem Informasi ITHB - Modern Web Platform

Selamat datang di repositori _source code_ untuk _website_ profil Sistem Informasi Institut Teknologi Harapan Bangsa (ITHB). Proyek ini dibangun untuk mewakili semangat inovasi digital _Professional Gen-Z_ dengan menggabungkan performa web modern dan elemen visual interaktif. 🚀

## ✨ Fitur Utama Terobosan

- **Interactive 3D Coin Engine:** _Hero section_ dilengkapi dengan koin 3D berbahan _silver_ realistis yang menampilkan logo HIMA SI dan ITHB di kedua sisinya. Koin ini mendeteksi interaksi _mouse_ (dapat ditarik dan diputar 360 derajat secara bebas) dan dilengkapi dengan sistem fisika _snap-back_ pegas yang akan mengembalikannya ke posisi semula secara elegan saat dilepas.
- **Next-Gen UI/UX:** Mengusung konsep desain _Glassmorphism_ gelap dengan aksen _neon glow_ khas _cyber/tech_, memberikan kesan futuristik namun tetap profesional.
- **Performa Optimal:** Telah dimigrasikan secara penuh ke ekosistem Next.js (App Router) untuk menjamin kecepatan _loading_ yang instan dan SEO yang kuat.
- **Komponen Modular:** Dibangun menggunakan arsitektur komponen dari Shadcn UI yang mudah disesuaikan dan dikembangkan di masa depan.

## 🛠️ Teknologi di Balik Layar

- **Core Framework:** [Next.js](https://nextjs.org/)
- **Styling Engine:** [Tailwind CSS v4](https://tailwindcss.com/)
- **3D Rendering:** [Three.js](https://threejs.org/) & [@react-three/fiber](https://docs.pmnd.rs/react-three-fiber/getting-started/introduction) (beserta Drei Utilities)
- **UI Library:** [Shadcn UI](https://ui.shadcn.com/) (Radix Primitives)
- **Iconography:** [Lucide React](https://lucide.dev/)

## 🚀 Cara Menjalankan Proyek Secara Lokal

Jika kamu ingin menjalankan atau berkontribusi pada proyek ini di komputermu sendiri, ikuti langkah-langkah berikut:

1. **Clone repositori ini:**

   ```bash
   git clone https://github.com/koko1514/si-web-next-v1.git
   ```

2. **Masuk ke dalam direktori proyek:**

   ```bash
   cd si-web-next-v1
   ```

3. **Install semua dependensi yang dibutuhkan:**

   ```bash
   npm install
   ```

4. **Jalankan server _development_:**

   ```bash
   npm run dev
   ```

5. Buka `http://localhost:3000` di _browser_ favoritmu untuk melihat hasilnya!
